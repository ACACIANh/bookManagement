1. 도서 관리 프로그램

2.1 데이터 베이스
 => 도서 정보 : 분류번호, 제목, 저자, 출판사, 분류, 상태

3.1 구현 기능
 1) 도서정보 저장
  => 분류번호, 제목, 저자, 출판사 입력받아 저장
  => 입력값에 대한 유효성 검사
  => 입력 분류번호에 대한 중복 검사

 2) 도서정보 변경
  => 분류번호를 입력받아 도서정보 검색 : 검색된 분류번호가 없는 경우 메세지 출력
  => 제목, 저자, 출판사를 입력받아 변경
  => 입력값에 대한 유효성 검사

 3) 도서정보 삭제
  => 분류번호를 입력받아 도서정보 삭제: 삭제할 도서정보가 없는 경우 메세지 출력
  => 입력값에 대한 유효성 검사

 4) 도서정보 검색
  => 제목을 입력받아 도서정보 검색 : 검색된 도서정보가 없는 경우 메세지 출력
  => 입력값에 대한 유효성검사
  ==> 저장된 모든 도서정보 검색


======================================================

2.2 데이터 베이스
 => 회원 정보 : 고유번호, 이름, 권한, 대출상태, 계정상태, 가입일, 전화번호, 주소

3.2 
 1) 회원정보 저장
  => 이름, 전화번호, 주소, 비밀번호 입력받아 저장
  => 입력 당시 관리자 로그인 여부에 따라 권한 부여
  => 입력값에 대한 유효성 검사
  => 입력 고유ID에 대한 중복 검사 ( 고려할 사항 많음 )
  => 비밀번호 추가 처리

 2) 회원정보 변경
  => 관리자 권한이 있을 경우 사용 가능
  => 회원ID를 입력받아 회원정보 검색 : 검색된 분류번호가 없는 경우 메세지 출력
  => 전화번호, 주소를 입력받아 변경
  => 입력값에 대한 유효성 검사

 3) 회원정보 삭제
  => 관리자 권한이 있을 경우 사용 가능
  => 분류번호를 입력받아 회원정보 삭제: 삭제할 회원정보가 없는 경우 메세지 출력
  => 입력값에 대한 유효성 검사

 4) 회원정보 검색
  => 이름을 입력받아 회원정보 검색 : 검색된 회원정보가 없는 경우 메세지 출력
  => 입력값에 대한 유효성검사
  ==> 저장된 모든 회원정보 검색
